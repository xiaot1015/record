### php extension swoole 
swoole 是一个异步多线程的php扩展，异步TCP/UDP，异步mysql，异步redis，数据连接池，消息队列，毫秒定时器，文件读写

* [swoole 文档](http://www.swoole.com/)

linux swoole 安装方法
* 通过pecl 安装   
```bash```
apt-get install php-pear
pecl install swoole
```

#### swoole 服务 模拟彩票
** 简介
模拟彩票，通过选择彩票类型，系统根据对应类型的彩票规则 模拟出来一个号码
